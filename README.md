# Playwright Framework (JavaScript)

## Setup
1. Install dependencies:
   npm install

2. Run tests:
   npx playwright test

## Notes
- This project is a **JavaScript** Playwright Test setup (no TypeScript).
- Config is in `playwright.config.js`.
- Tests live in `tests/`, page objects in `pages/`.
- Credentials are read from environment variables (e.g. `PAYROLL_USER`, `PAYROLL_PASS`).
